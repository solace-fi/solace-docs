


## Functions
### stakes
```solidity
  function stakes(
    address account
  ) external returns (uint256)
```
function returns the staking amount of the address.


#### Parameters:
| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
|`account` | address | The address of the account.

#### Return Values:
| Name                           | Type          | Description                                                                  |
| :----------------------------- | :------------ | :--------------------------------------------------------------------------- |
|`amount`| address | The amount of stake.

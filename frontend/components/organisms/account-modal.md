## Props

### `AccountModalProps`

| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
| `isOpen` | boolean | Boolean to indicate whether the modal is open or not.
| `closeModal` | () => void | Function to close the modal.
